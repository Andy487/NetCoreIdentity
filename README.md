# ASP.NET Core Identity Authentication

A sample ASP.NET Core 8 Razor Pages application demonstrating:

- User Registration
- User Login
- User Logout
- Cookie Authentication
- Authorization
- ASP.NET Core Identity
- Entity Framework Core
- SQL Server
- Role-Based Authorization
- Claims-Based Authorization
- Password Hashing
- Identity User Management

## Technologies

- ASP.NET Core 8
- Razor Pages
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- Bootstrap 5

## Features

- Register User
- Login
- Logout
- Protected Dashboard
- Role Management
- Authorization
- Identity Authentication

## Database

SQL Server LocalDB using Entity Framework Core Migrations.
